# B2B / Professional Tier — Staged Source (NOT shipped)

This folder holds **industrial / municipal operations** material drafted by the Gemini
research agent. It is **parked here on purpose** — it is *not* linked from the live
citizen site and *not* part of the Phase 1 Academy.

## Why parked

Recyclopedia's confirmed priority is **citizens first** (everyday people with a
smartphone). The content below targets a different audience — facility operators,
municipalities, vendors — which is a **later tier** (B2B / professional certification).
We keep it version-controlled and corrected so it's ready the day we build that tier.

## Contents

- `vendor-sla.md` — Near-Infrared (NIR) optical sorting Service Level Agreement template.
- `operations-reference.md` — Technical risk register (NIR failure modes) + the organics
  diversion financial cost-benefit analysis.
- `frontline-multilang.md` — Frontline sorter SOP checklists, visual wall posters, and
  the contamination incident form, in EN / ES / PT / FR. **Localization bugs fixed**
  (see PROVENANCE.md item 4).
